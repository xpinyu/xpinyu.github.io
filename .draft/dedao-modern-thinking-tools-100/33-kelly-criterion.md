# 凯利公式：乘法世界里的认知变现

## 三句摘要

- 这讲用 `凯利公式（Kelly criterion）` 解决“有了高价值信息之后该押多大”的问题。它不是保守的投资安全法则，而是把认知优势翻译成行动仓位的机器。
- 凯利公式的核心不是胆量，而是 `edge`：你对局面的判断比市场共识高明多少。没有 edge 就不要下注；edge 很好却不出手，也是在浪费认知优势。
- 作者把凯利公式推广到职业、时间、健康和信任：人生很多资产都是乘法式复利资产，正确策略不是 all-in，也不是永不下注，而是认知先行、控制仓位、追求长期复合增长，并永远避免清零。

## 这讲的核心诊断

前一讲说了信息价值：信息只有能改变行动，才有价值。

这一讲继续追问：

- 当你真的有价值信息时，应该投入多大？

很多人的问题不是信息不足，而是仓位错配：

- 为几毛钱股票波动高杠杆杀进杀出
- 面对真正重要的职业机会却犹豫不决
- 小利忘命，大事惜身

作者借曹操评价袁绍的话概括：

- 色厉胆薄，好谋无断；干大事而惜身，见小利而忘命。

这不是信息问题，而是仓位问题。

## 凯利公式的来历

约翰·凯利是贝尔实验室科学家。

他 1956 年的论文原本研究通信问题：

- 信息传递有噪声
- 噪声不能完全消除
- 但有错误率的信息仍然有用

凯利设想一个赌徒通过有噪声的电话线接收赛马内幕消息：

- 电话那头 100% 知道哪匹马会赢
- 但电话线有噪声
- 赌徒听到的结果有概率是错的

问题是：

- 他该下注多少？

如果 all-in，听错一次就破产。

如果下注太少，又浪费信息优势。

凯利证明：存在一个最佳下注比例，让资金长期对数增长率最大化，而且这个最大增长率在数学上等同于电话线的信息传输率。

所以：

- 信息论和复利增长被连接起来了

作者总结：

- `你的增长上限，受限于你的认知带宽。`

## 凯利公式解决什么问题

凯利公式适用于：

- 不确定
- 乘法式结算
- 可重复
- 有输有赢
- 输赢会影响下一轮本金

它解决的不是“怎么赢一把”，而是：

- `怎么一直赢`

在长期重复的乘法世界里，仓位会决定你能否活下来并复利增长。

## 公式的直觉

公式中的几个变量：

- `p`：赢的概率
- `q = 1 - p`：输的概率
- `b`：赔率，下注 1 元赢了净赚多少
- `f*`：最佳下注比例

例如：

- 胜率 p = 0.6
- 赔率 b = 1
- 凯利公式给出的最佳下注比例是 20%

重点是：

- 下注大小不是看你多想赢
- 也不是看你胆子多大
- 而是看胜率和赔率

## Edge：认知优势才是核心

赌场黑话里，凯利公式可以理解为：

- `f* = edge / odds`

其中 `odds` 是赔率，是市场给的。

真正重要的是 `edge`：

- 每下注 1 元的期望净利润
- 也就是你的认知优势

你之所以敢下注，是因为你认为真实胜率比市场估计更高。

作者强调：

- 凯利公式让你赚的，是你超出大众认知的钱。
- 你比大众高明多少，凯利就帮你赚多少，但不能更多。

如果 edge 不大于 0，就不要下注。

说白了：

- 你要是不比别人高明，就别玩。

## 袁绍错过巨大的 edge

作者用袁绍举例。

建安五年，曹操讨伐刘备，许都空虚。谋士田丰建议袁绍立即出兵许都，这是巨大战略窗口。

按凯利公式说：

- 这是一个巨大的 edge

但袁绍因为小儿子生病，心烦不愿出兵。

结果曹操回兵，官渡之战袁绍失败。

作者的判断很重：

- `如果 edge 好，出手是一种义务。`

这说明，保守不一定是稳健；面对高 edge 不下注，也是一种错误。

## 生物学中的凯利：赌注对冲

作者提到 `表型赌注对冲（Phenotypic Bet-Hedging）`。

一些单细胞生物面对环境波动，不会让所有个体同时繁殖，而是：

- 一部分进入休眠
- 其余个体出去繁殖

这样即使环境恶化，族群也不会全灭。

研究发现，进入休眠的比例在数学上等价于凯利公式中的未下注比例。

这说明：

- 生物多样性可能是系统面对不确定性时的凯利仓位

## 人生的三个凯利原则

### 1. 别把自己清零

只要 p 不是严格等于 1，就不要 all-in。

而贝叶斯主义者永远不会把概率设成 0 或 1。

再考虑到估计误差，就更要留余地。

诱惑再大，也要保本，确保自己留在牌桌上。

### 2. 认知先行

先问：

- 我有多大 edge？

再问：

- 我该下多大注？

很多人反过来：

- 先有下注冲动
- 再到处找理由证明自己有优势

作者提醒：

- 我们是把认知变现，不是用认知解释变现。

巴菲特不买自己看不懂的股票，正是这个原则。

### 3. 追求复利，不是赢一把

凯利公式的设定是连续玩很多把。

它关心的是长期积累最大化，而不是单次胜负。

只要策略正确，单次成败都不重要。

## 职业规划中的凯利

职业是一种乘法资产：

- 技能可积累
- 声誉可积累
- 网络可积累

所以换工作、换职业也可以用凯利思考。

流程是：

1. 先问 edge：我在新方向上是否真的比别人强，是否学得更快、交付更好、更愿意熬？
2. 再看 odds：这个方向的回报结构和上限如何？
3. 最后谈仓位：如果 edge 有但不确定，就用半凯利或分数凯利试水。

例如：

- 副业
- 小项目
- 兼职尝试
- 低成本验证

在凯利视角下，“求稳”和“求拼”不是二分，而是仓位大小。

## 时间和精力中的凯利

有效精力和健康也是资产。

你每天把时间分配到：

- 工作
- 学习
- 运动
- 社交
- 娱乐

这些都像下注。

重要的不是某件事听起来有没有价值，而是：

- 在你当前状态下，它有没有 edge？

如果你已经很累，熬夜完成一个重要工作是否值得，要看边际收益和损失。

时间管理的凯利策略是：

- 少出手
- 把仓位集中到能产生复利的少数变量上

## 信任中的凯利

信任也是乘法资产：

- 一次次小合作积累
- 一次重大背叛清零

人际关系的凯利策略：

1. 刚认识时小下注，交给对方小事。
2. 表现好后逐步加仓。
3. 始终确保潜在收益配得上潜在损失。
4. 永远保留止损机制，不交出会让你瞬间破产的权限。

这不是主张所有关系都公式化，而是提醒：行走江湖要管理信任资产。

## 勇敢是什么

作者最后重新定义勇敢。

如果凯利公式建议押 20%，你押 25%，这未必是勇敢，可能只是冲动或无知。

大多数人被单次输赢左右：

- 输一把就胆怯
- 赢一把就狂妄

真正的勇敢可能是：

- 不被暂时输赢左右
- 清醒地按正确仓位出手
- 克服本能
- 保留下一次下注能力

人生的根本自由，是你始终还有下一次下注。

## 论证链

1. 有价值信息之后，关键问题变成该投入多大。
2. 凯利公式把有噪声的信息优势转化成最佳下注比例。
3. 核心变量不是胆量，而是 edge 和 odds。
4. 没有 edge 不下注，有大 edge 不下注也是错误。
5. 人生中的职业、时间、健康、信任都具有乘法资产特征。
6. 凯利原则要求避免清零、认知先行、追求复利。
7. 真正的勇敢是按正确仓位长期行动，而不是单次冲动。

## 关键例子

- 有噪声的赛马电话线：说明信息优势如何转化为下注比例。
- 袁绍错失许都：说明高 edge 不出手也是错误。
- 表型赌注对冲：说明凯利思想也出现在生物进化中。
- 职业副业试水：说明分数凯利可用于人生选择。
- 信任管理：说明关系也需要小下注、加仓和止损。

## 反直觉点

- `凯利公式不是保守法则，而是认知变现机器。`
- `押太大和不敢押都是错的。`
- `没有 edge 就别玩，有 edge 就该出手。`
- `勇敢不是多押，而是不被输赢带偏地按仓位押。`
- `人生自由是始终还有下一次下注的能力。`

## 给自己的应用版本

- 面对机会，先问 edge，不先问我想不想。
- 对不确定但有 edge 的机会，用小项目、分数凯利试水。
- 禁止 all-in 到会清零的程度，包括钱、健康、声誉和关系。
- 把时间集中到能复利的变量上，减少低 edge 的频繁出手。
- 在信任关系中，小下注验证，再逐步加仓。

## 可继续追问的问题

- 我现在认为自己有 edge 的地方，证据是什么？
- 我是不是在小利上忘命，在大事上惜身？
- 哪些资产一旦清零，我承受不起？
- 这个机会适合全仓、半仓、小仓，还是完全不下注？

## 复习卡

- 凯利公式：把认知优势翻译成最佳下注比例。
- `edge` = 你超出大众认知的优势。
- 三原则：`别清零 / 认知先行 / 追求复利`。
- 无 edge 不下注；大 edge 不出手也是错。
- 勇敢 = 清醒地按正确仓位行动。
