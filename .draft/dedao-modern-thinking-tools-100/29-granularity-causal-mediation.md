# 颗粒度和因果中介：用模型思考

## 三句摘要

- 这讲讲决策之前的认识论：要想控制一个事物，必须先在头脑里建立它的模型。没有模型，人只能对信号冲动反应；有模型，才知道哪些变量、机制和按钮真正起作用。
- 好模型至少有两个要求：第一，颗粒度恰到好处，既不过拟合也不欠拟合；第二，包含因果关系，特别是能找到真正可干预的中介变量。理解不是堆细节，而是高质量压缩。
- 作者用“生气会不会导致癌症”、《点球成金》、足球期望威胁、学习成绩和公司管理说明：高手不会只盯终点，而会寻找改变概率结构的关键中介。

## 这讲的核心诊断

网上常有一种市侩哲学：

- 癌症是气出来的
- 想不得癌，就要内心强大到混蛋
- 不要憋着，不要拧巴

作者直接说：这是伪科学。

美国癌症协会和美国国家癌症研究所都指出，人的个性、想法、情绪与癌症发生或复发之间没有清晰直接链条。多项大规模研究也显示，在排除吸烟、饮酒等因素后，压力或负面情绪与主要癌症风险没有明确关联。

但这并不等于压力对健康无害。

更精确的模型是：

- 压力不直接导致癌症
- 压力可能导致不良生活习惯
- 不良生活习惯可能增加癌症风险

这就是本讲的核心：你必须有模型，尤其要找出中介变量。

## 什么是模型

模型是事物在头脑中的结构镜像。

作者用宿舍管理员举例：

- 没有模型时，一听见吵闹就大喊大叫或挨个敲门。
- 有模型时，知道 305 有低音炮、117 打联网游戏、二楼水泵坏了，于是可以精准干预。

生活中很多人没有模型：

- 体检指标差了就喝枸杞
- 项目延期就要求加班
- 孩子成绩下滑就打骂

他们只是在对信号做大动作。

有模型，才知道事情怎么运行，控制点在哪里。

## 好调节器定理

作者引用控制论里的 `好调节器定理（Good Regulator Theorem）`：

- 系统的每一个好调节器，都必须是该系统的一个模型。

它的意思是：

- 你的认知结构必须同构于你想控制的对象
- 模型太简单，就会丢失关键信息
- 模型太错，决策必然错

作者总结为：

- `你能理解到什么程度，才能控制到什么程度。`

## 好模型要求一：颗粒度恰到好处

模型的复杂度，也就是职场常说的 `颗粒度（Granularity）`，必须合适。

太粗糙的问题：

- 把癌症解释成“气出来的”
- 把学习不好解释成“不努力”
- 把公司业绩差解释成“员工不拼”

太复杂的问题：

- 把所有变量全列出来
- 信息量太大，算力不够
- 注意力被噪音占满

模型必须是对现实的压缩。

作者引入 `最小描述长度（Minimum Description Length, MDL）`：

- 模型本身的描述长度
- 用这个模型解释现实时需要补充的例外说明

两者之和越短，模型越好。

换句话说：

- 好模型能用最短代码解释最多数据

## 过拟合与欠拟合

作者用天气举例：

- 把过去一年每天的天气都描述一遍，是过拟合。
- 只说“这地方总下雨”，是欠拟合。
- 说“温带海洋性气候，受西风带控制，全年温和湿润，阴雨多但多是毛毛雨，很少暴雨”，就是好模型。

好模型有：

- 规律
- 机制
- 压缩
- 可迁移解释力

这就是作者说的：

- `理解即压缩（Understanding is Compression）`

高手所谓“半秒看透本质”，不是神秘直觉，而是能把大量噪音压缩成高质量模型。

## 好模型要求二：包含因果关系

只有相关性还不够，模型要包含因果。

作者借朱迪亚·珀尔的思想说明：我们关心因果，不是为了哲学上证明世界有绝对因果，而是为了知道：

- 我做什么干预，能改变这件事发生的概率？

如果强行介入，让 X 发生，而 Y 的概率随之增加，才可以说 X 对 Y 有因果效应。

例如：

- 公鸡打鸣和太阳升起是相关，不是因果。
- 按开关灯亮，是可干预的因果。

好模型应该能把变量用因果箭头串起来。

## 中介变量：真正的控制点

任何模型都不能穷尽所有因果关系。关键是找到足够好的颗粒度，尤其是找到 `中介变量（Mediator）`。

“生气 -> 癌症”太粗糙。

更精确的是：

- 生气 / 长期压力
- 导致不良生活习惯
- 不良生活习惯提高癌症风险

中介变量是“不良生活习惯”。

这就让问题可操作：

- 如果情绪差，但仍然运动、控制饮食、不吸烟不酗酒，那么风险机制就被切断。

有了中介，才有控制点。

## 《点球成金》：因果模型打败直觉

作者用《点球成金》说明模型如何改变决策。

传统棒球球探的模型是：

- 球星 -> 赢球

他们喜欢买：

- 能打全垒打的人
- 长得像球星的人
- 姿势优美的人

这些人很贵。

奥克兰运动家队总经理比利·比恩因为没钱，建立了更好的模型：

- 击球 + 选球眼光 -> 上垒 -> 得分 -> 赢球

`上垒` 是被忽视的关键中介变量。

那些长得丑、姿势怪、不会打全垒打但特别能上垒的人，被市场低估，于是可以买得很便宜。

这不是简单的大数据胜利，而是因果模型对直觉的降维打击。

## 足球里的期望威胁

作者把点球成金思路迁移到足球。

足球中真正有价值的不一定是射门或助攻，而可能是：

- 倒数第二脚传球
- 倒数第三脚传球
- 带球突破

这些动作改变了场上的概率结构。

例如一脚传球把球从低威胁区域送到高威胁区域，让进球概率从 2% 提升到 30%。

现代足球模型称之为：

- `期望威胁（Expected Threat, xT）`

作者的关键提醒是：

- `别盯终点，盯中介。`

## 日常应用

### 买手机

模型太短：

- 我就要最贵的
- 我买便宜的就行

模型太长：

- 研究所有参数、评测、价格，迟迟不买

好模型是先问用途：

- 拍娃还是拍夜景？
- 打游戏还是刷视频？
- 最关键变量是哪几个？

### 学习成绩

不要只说“不努力”。

一个简单模型是：

- 时间 + 注意力 -> 记忆编码 -> 提取 -> 成绩

不同环节出问题，干预方式不同：

- 时间和注意力不足，才是努力问题
- 编码差，可能是材料或讲解问题
- 提取差，可能是练习不足

### 公司管理

只盯收入，是欠拟合。

把所有变量都日报，是过拟合。

好管理要识别核心中介：

- 留存
- 转化率
- 渠道质量
- 产品体验
- 交付速度
- 团队协作
- 决策效率

然后找到一两个最关键中介干预。

## 模型的边界

作者提醒：

- 没有绝对标准的颗粒度
- 颗粒度由你的主观需求决定
- 现场总会有意外

地图不是疆域。

模型永远不是现实本身，但没有地图，你连路都上不了。

## 论证链

1. 错误决策常来自没有模型，只对表面信号反应。
2. 好调节器必须是系统的模型，控制依赖理解。
3. 好模型要有恰当颗粒度，既不过拟合也不欠拟合。
4. 好模型还要包含可干预的因果关系。
5. 关键是找到中介变量，因为中介变量才是控制点。
6. 点球成金和期望威胁说明，高手盯的是改变结果概率的中间环节。
7. 因此，用模型思考就是用压缩和因果找到可操作的杠杆。

## 关键例子

- 生气与癌症：说明直接因果太粗糙，关键是生活习惯中介。
- 宿舍管理员：说明模型让干预从粗暴变精准。
- 天气描述：说明好模型是高质量压缩。
- 点球成金：说明上垒是被低估的因果中介。
- 期望威胁：说明真正关键的动作可能不是终点动作。
- 学习成绩模型：说明不同环节对应不同干预。

## 反直觉点

- `理解不是知道更多细节，而是更好地压缩。`
- `没有模型的数据，不会自动告诉你答案。`
- `相关性不是因果，可干预性才是因果的实用标准。`
- `终点变量往往太晚，真正的控制点在中介变量。`
- `地图不是疆域，但没有地图你连路都上不了。`

## 给自己的应用版本

- 遇到问题先画因果链，不急着下结论。
- 每个复杂问题都问：这里的关键中介变量是什么？
- 检查模型是否过短：是不是只剩一句口号？
- 检查模型是否过长：是不是堆了太多无法行动的细节？
- 做管理、学习、健康和投资决策时，不只盯结果指标，要找改变概率结构的前置指标。

## 可继续追问的问题

- 我现在对这个问题的模型是什么？
- 这个模型是欠拟合、过拟合，还是恰到好处？
- 我抓住的是终点变量，还是中介变量？
- 哪个变量一旦被干预，最可能改变结果概率？

## 复习卡

- 模型 = 事物在头脑中的结构镜像。
- 好模型两要求：`合适颗粒度 / 包含因果关系`。
- 最小描述长度：用最短模型解释最多数据。
- 中介变量 = 真正可干预的控制点。
- 口诀：别盯终点，盯中介。
