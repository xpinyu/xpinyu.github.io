## 学习顺序

1. [`00-preface.md`](./00-preface.md)
   定义整门课的问题意识：不是更努力，也不是更世故，而是要建立一套可自我纠错的现代思维工具系统。

2. [`01-narrative.md`](./01-narrative.md)
   说明世界不是“事实堆”，而是“叙事场”；叙事决定行动、意义与公共协作。

3. [`02-heavy-tail.md`](./02-heavy-tail.md)
   解释为什么现代世界不是平均分配，而是极端涌现；关键是跳出加法世界，进入乘法世界。

4. [`03-agency.md`](./03-agency.md)
   拆解旧时代留下来的稳态生存逻辑，说明为什么现代人必须做能动者，而不是旧叙事的执行器。

5. [`04-constraints.md`](./04-constraints.md)
   给前面三讲加上现实校正：行动之前先盘硬约束，成熟不是许愿，而是约束内找出路。

6. [`05-qa-narrative.md`](./05-qa-narrative.md)
   一次跨讲问答整理，把叙事、重尾、AI 使用和教育判断这些抽象框架拉回具体场景。

7. [`06-possibility.md`](./06-possibility.md)
   说明不确定性不是人生故障，而是意义燃料；关键不是消灭不确定，而是分类、管理和利用它。

8. [`07-core-self.md`](./07-core-self.md)
   把“自我”拆成进程、界面和内核三层，强调真正的成长是重写底层模型参数。

9. [`08-capability-seeking.md`](./08-capability-seeking.md)
   进入成长战略模块：与其盯死某个局部奖励，不如持续为自己增加选项、能耐和赋能。

10. [`09-supply-side.md`](./09-supply-side.md)
   说明现代竞争的本质不是抢占资源，而是争取更高质量合作网络中的被需要资格。

11. [`10-compound-interest.md`](./10-compound-interest.md)
   纠正对复利的鸡汤式误解，把“积累优势”从金钱扩展到多种资本和长期飞轮。

12. [`11-qa-admit-wrong.md`](./11-qa-admit-wrong.md)
   一次跨讲问答整理，核心聚焦在模型更新为何困难，以及如何用最小试探、最小胜利和身份解离推动改变。

13. [`12-self-determination.md`](./12-self-determination.md)
   说明真正的一流人物不靠痛苦硬卷，而靠高质量动机、自主感、胜任感和关系感来长久驱动。

14. [`13-special-qa-notes.md`](./13-special-qa-notes.md)
   对“基本世界观”模块的直播复盘，把叙事权、好运、元认知、高波动与 AI 时代的人类稀缺性串在一起。

15. [`14-free-energy.md`](./14-free-energy.md)
   用“惊讶最小化”和“活着就是对齐”解释生命、学习、拖延、习惯与关系的共通机制。

16. [`15-attention-pro-mode.md`](./15-attention-pro-mode.md)
   把注意力从“自律问题”提升为稀缺战略资源，说明真正的专注来自高认知负荷与主动建模。

17. [`16-woop.md`](./16-woop.md)
   提供一个把愿望编译成行动代码的四步流程，用 `If-Then` 结构打破默认设置与日常漂流。

18. [`17-social-capital.md`](./17-social-capital.md)
   说明向上流动不仅取决于个人能力，还取决于你所处的位置、跨阶层连接与结构洞角色。

19. [`18-qa-starting-liked-things.md`](./18-qa-starting-liked-things.md)
   一次跨讲问答整理，核心聚焦在中年积累是否作废、热爱与人设的区别，以及为什么行动往往先于内化。

20. [`19-cognitive-decoupling.md`](./19-cognitive-decoupling.md)
   把负面情绪调节拆成三步流程：先拆开叙事与事实，再调用视角，最后重写意义框架。

21. [`20-identity.md`](./20-identity.md)
   讨论身份认同如何作为行为底层代码运行，并说明成熟的关键是把身份从“主体”降级成可主动调用的“客体”。

22. [`21-security.md`](./21-security.md)
   说明安全感不是软情绪，而是探索、创造与高质量关系的前提，并用“安全基地/安全港湾”解释其来源。

23. [`22-track-selection.md`](./22-track-selection.md)
   用“天兵天将 vs 孙悟空”区分体制内优化与体制外创造两种路径，强调先选游戏，再谈努力。

24. [`23-field.md`](./23-field.md)
   引入布迪厄的场域理论，说明个人成败常取决于是否理解并适配该场域的计分规则、资本与象征权力。

## 贯穿线索

- 这套课的底层目标不是“会几个概念”，而是获得一套能稳定更新的世界模型。
- 从 `叙事 -> 重尾 -> 能动 -> 约束 -> 可能 -> 内核 -> 能耐 -> 供给侧 -> 复利 -> 动机 -> 对齐 -> 注意力 -> 执行 -> 位置 -> 情绪调节 -> 身份 -> 安全 -> 赛道 -> 场域`，其实是在回答更完整的一串问题：
- 我们是如何理解世界的？
- 世界真实长什么样？
- 在这样的世界里我该以什么姿态行动？
- 行动时哪些边界绝不能假装不存在？
- 面对不确定性，我该逃避还是利用？
- 作为行动者，我自己到底由什么构成？
- 我应该优先追逐即时奖励，还是追逐选项与赋能？
- 在社会协作里，我该先盯需求，还是先练供给？
- 我的优势如何从一次性表现，变成可积累的复利？
- 我为什么会努力，什么样的动机才可持续？
- 当我与世界不匹配时，究竟该改自己，还是改环境？
- 我怎样把注意力从被动消费，转成主动建模？
- 我怎样把愿望编译成真正可执行的行动？
- 在一个不均匀分布的世界里，我该站到什么位置上？
- 当情绪升起时，我怎样不被自己的第一反应接管？
- 我如何运用身份而不是被身份所绑架？
- 人为什么需要依靠，安全感又如何决定探索半径？
- 我到底该进入哪一种游戏结构？
- 一个地方真正奖励的到底是什么，我又该如何识别它？
