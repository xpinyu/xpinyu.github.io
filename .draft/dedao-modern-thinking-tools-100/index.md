## 学习顺序

1. [`00-preface.md`](./00-preface.md)
   定义整门课的问题意识：不是更努力，也不是更世故，而是要建立一套可自我纠错的现代思维工具系统。

2. [`01-narrative.md`](./01-narrative.md)
   说明世界不是“事实堆”，而是“叙事场”；叙事决定行动、意义与公共协作。

3. [`02-heavy-tail.md`](./02-heavy-tail.md)
   解释为什么现代世界不是平均分配，而是极端涌现；关键是跳出加法世界，进入乘法世界。

4. [`03-agency.md`](./03-agency.md)
   拆解旧时代留下来的稳态生存逻辑，说明为什么现代人必须做能动者，而不是旧叙事的执行器。

5. [`04-constraints.md`](./04-constraints.md)
   给前面三讲加上现实校正：行动之前先盘硬约束，成熟不是许愿，而是约束内找出路。

6. [`05-qa-narrative.md`](./05-qa-narrative.md)
   一次跨讲问答整理，把叙事、重尾、AI 使用和教育判断这些抽象框架拉回具体场景。

7. [`06-possibility.md`](./06-possibility.md)
   说明不确定性不是人生故障，而是意义燃料；关键不是消灭不确定，而是分类、管理和利用它。

8. [`07-core-self.md`](./07-core-self.md)
   把“自我”拆成进程、界面和内核三层，强调真正的成长是重写底层模型参数。

9. [`08-capability-seeking.md`](./08-capability-seeking.md)
   进入成长战略模块：与其盯死某个局部奖励，不如持续为自己增加选项、能耐和赋能。

10. [`09-supply-side.md`](./09-supply-side.md)
   说明现代竞争的本质不是抢占资源，而是争取更高质量合作网络中的被需要资格。

11. [`10-compound-interest.md`](./10-compound-interest.md)
   纠正对复利的鸡汤式误解，把“积累优势”从金钱扩展到多种资本和长期飞轮。

12. [`11-qa-admit-wrong.md`](./11-qa-admit-wrong.md)
   一次跨讲问答整理，核心聚焦在模型更新为何困难，以及如何用最小试探、最小胜利和身份解离推动改变。

13. [`12-self-determination.md`](./12-self-determination.md)
   说明真正的一流人物不靠痛苦硬卷，而靠高质量动机、自主感、胜任感和关系感来长久驱动。

14. [`13-special-qa-notes.md`](./13-special-qa-notes.md)
   对“基本世界观”模块的直播复盘，把叙事权、好运、元认知、高波动与 AI 时代的人类稀缺性串在一起。

15. [`14-free-energy.md`](./14-free-energy.md)
   用“惊讶最小化”和“活着就是对齐”解释生命、学习、拖延、习惯与关系的共通机制。

16. [`15-attention-pro-mode.md`](./15-attention-pro-mode.md)
   把注意力从“自律问题”提升为稀缺战略资源，说明真正的专注来自高认知负荷与主动建模。

17. [`16-woop.md`](./16-woop.md)
   提供一个把愿望编译成行动代码的四步流程，用 `If-Then` 结构打破默认设置与日常漂流。

18. [`17-social-capital.md`](./17-social-capital.md)
   说明向上流动不仅取决于个人能力，还取决于你所处的位置、跨阶层连接与结构洞角色。

19. [`18-qa-starting-liked-things.md`](./18-qa-starting-liked-things.md)
   一次跨讲问答整理，核心聚焦在中年积累是否作废、热爱与人设的区别，以及为什么行动往往先于内化。

20. [`19-cognitive-decoupling.md`](./19-cognitive-decoupling.md)
   把负面情绪调节拆成三步流程：先拆开叙事与事实，再调用视角，最后重写意义框架。

21. [`20-identity.md`](./20-identity.md)
   讨论身份认同如何作为行为底层代码运行，并说明成熟的关键是把身份从“主体”降级成可主动调用的“客体”。

22. [`21-security.md`](./21-security.md)
   说明安全感不是软情绪，而是探索、创造与高质量关系的前提，并用“安全基地/安全港湾”解释其来源。

23. [`22-track-selection.md`](./22-track-selection.md)
   用“天兵天将 vs 孙悟空”区分体制内优化与体制外创造两种路径，强调先选游戏，再谈努力。

24. [`23-field.md`](./23-field.md)
   引入布迪厄的场域理论，说明个人成败常取决于是否理解并适配该场域的计分规则、资本与象征权力。

25. [`24-qa-deep-or-move.md`](./24-qa-deep-or-move.md)
   一次跨讲问答整理，核心聚焦如何判断该深耕还是挪位置，并把 AI、信任、认知重评、身份、安全感和赛道问题拉回结构判断。

26. [`25-exploration-exploitation.md`](./25-exploration-exploitation.md)
   用探索与利用、多臂老虎机、吉廷斯指数和超级老年人研究，说明年轻不是年龄，而是系统持续更新的频率。

27. [`26-resonance.md`](./26-resonance.md)
   作为成长战略模块收束，引入罗萨的共鸣理论，说明高级生活不是占有更多资源，而是与世界建立有回应的关系。

28. [`27-no-free-lunch.md`](./27-no-free-lunch.md)
   开启决策判断模块，用无免费午餐定理说明不存在通用最优算法，任何决策都必须先设定偏置和价值函数。

29. [`28-probability-distribution.md`](./28-probability-distribution.md)
   把决策从选择单个结果改写成选择概率分布，强调均值、方差、尾部风险、稳健性和过程质量。

30. [`29-granularity-causal-mediation.md`](./29-granularity-causal-mediation.md)
   用模型、颗粒度和因果中介解释如何从表面结果回到可干预的结构变量，避免欠拟合和过拟合。

31. [`30-qa-mission-calling.md`](./30-qa-mission-calling.md)
   一次跨讲问答整理，把僵化场域中的自处、使命的定义、AGI 的偏置、发愿与决策的区别讲清楚。

32. [`31-bayesian-prior.md`](./31-bayesian-prior.md)
   引入贝叶斯主义和先验，说明判断虽然主观，但可以通过证据持续更新，避免无先验和先验不可动摇两种错误。

33. [`32-value-of-information.md`](./32-value-of-information.md)
   用信息价值理论区分沙子和金子，强调只有能改变行动的信息才有决策价值。

34. [`33-kelly-criterion.md`](./33-kelly-criterion.md)
   用凯利公式把认知优势翻译成行动仓位，说明人生中的职业、时间、健康和信任都需要管理下注比例。

35. [`34-non-ergodicity.md`](./34-non-ergodicity.md)
   解释非遍历性如何让正期望游戏仍可能使个体破产，并给出少交易、杠铃策略、成为庄家和风险共担四个对策。

36. [`35-antifragility.md`](./35-antifragility.md)
   用反脆弱和非对称风险说明，有些系统从波动中受损，有些系统从波动中获益，关键是识别自己站在哪一边。

37. [`36-qa-best-model-ability.md`](./36-qa-best-model-ability.md)
   一次跨讲问答整理，围绕模型库存、儿童先验、学习系统、真实 edge 和非遍历成长，说明判断系统如何长期变强。

38. [`37-options.md`](./37-options.md)
   把期权从金融概念扩展成决策工具，说明如何用有限成本购买稀缺可选项，并在合适时机行权或承诺。

39. [`38-state-leverage.md`](./38-state-leverage.md)
   说明努力不是全部，关键是行动之后留下什么状态；用前置杠杆、顺序杠杆和约束杠杆减少返工、降低摩擦、打通瓶颈。

40. [`39-selection-bias.md`](./39-selection-bias.md)
   系统整理自我选择、幸存者偏差、分组选拔偏差和门槛偏差，提醒我们看到的真实样本未必代表真实世界。

41. [`40-regression-to-mean.md`](./40-regression-to-mean.md)
   用回归均值解释为什么极端表现之后常会自然回落或反弹，强调不要把正常波动误判成因果和管理成效。

42. [`41-prospect-theory.md`](./41-prospect-theory.md)
   用前景理论说明人不是按最终财富决策，而是围绕心理参照点判断得失；真正驱动冒险的常常不是贪婪，而是不甘。

43. [`42-qa-dsm-gantt.md`](./42-qa-dsm-gantt.md)
   一次跨讲问答整理，把反脆弱健康实践、DSM 与甘特图、自我校准、普通原因与特殊原因变异放回机制和反馈中判断。

44. [`43-reference-class.md`](./43-reference-class.md)
   引入参考类预测，用外部视角和同类分布校正规划谬误，提醒自己在预测时先当样本，再当主角。

45. [`44-superforecasting.md`](./44-superforecasting.md)
   把预测工程化为概率化和可检验流程，通过费米化、参考类、贝叶斯更新和布里尔分数训练自己的判断账本。

## 贯穿线索

- 这套课的底层目标不是“会几个概念”，而是获得一套能稳定更新的世界模型。
- 从 `叙事 -> 重尾 -> 能动 -> 约束 -> 可能 -> 内核 -> 能耐 -> 供给侧 -> 复利 -> 动机 -> 对齐 -> 注意力 -> 执行 -> 位置 -> 情绪调节 -> 身份 -> 安全 -> 赛道 -> 场域 -> 探索/利用 -> 共鸣 -> 偏置 -> 概率分布 -> 模型 -> 贝叶斯更新 -> 信息价值 -> 仓位 -> 非遍历 -> 反脆弱 -> 期权 -> 状态杠杆 -> 选择偏差 -> 回归均值 -> 前景理论 -> 参考类预测 -> 超级预测`，其实是在回答更完整的一串问题：
- 我们是如何理解世界的？
- 世界真实长什么样？
- 在这样的世界里我该以什么姿态行动？
- 行动时哪些边界绝不能假装不存在？
- 面对不确定性，我该逃避还是利用？
- 作为行动者，我自己到底由什么构成？
- 我应该优先追逐即时奖励，还是追逐选项与赋能？
- 在社会协作里，我该先盯需求，还是先练供给？
- 我的优势如何从一次性表现，变成可积累的复利？
- 我为什么会努力，什么样的动机才可持续？
- 当我与世界不匹配时，究竟该改自己，还是改环境？
- 我怎样把注意力从被动消费，转成主动建模？
- 我怎样把愿望编译成真正可执行的行动？
- 在一个不均匀分布的世界里，我该站到什么位置上？
- 当情绪升起时，我怎样不被自己的第一反应接管？
- 我如何运用身份而不是被身份所绑架？
- 人为什么需要依靠，安全感又如何决定探索半径？
- 我到底该进入哪一种游戏结构？
- 一个地方真正奖励的到底是什么，我又该如何识别它？
- 我该继续深耕，还是换一个能产生复利的位置？
- 我该如何在利用已有成果的同时继续探索？
- 高级生活为什么不是拥有更多资源，而是拥有更多共鸣？
- 如果没有通用最优算法，我如何设定自己的偏置？
- 我该如何看概率分布，而不是被单次结果绑架？
- 我怎样建立颗粒度合适、包含因果中介的模型？
- 我的先验该如何被证据更新？
- 哪些信息真的能改变行动，哪些只是焦虑消费？
- 当我有认知优势时，该押多大仓位？
- 作为个体，我该如何避免被非遍历风险清零？
- 面对波动，我是脆弱的一边，还是反脆弱的一边？
- 在不确定性中，我该何时保留选项，何时主动承诺？
- 我这一步行动会让下一步更容易，还是更困难？
- 我看到的样本是否真的代表世界，还是只是筛选后的局部？
- 我看到的极端结果是趋势变化，还是正常波动？
- 我当前的冒险和保守，是理性判断，还是参照点在驱动？
- 我是否该先把自己放进同类分布里，而不是相信主角叙事？
- 我能否把模糊判断写成概率，并用结果给自己打分？
